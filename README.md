# 🗡️ Tap Tap Loot Script
A script that automatically presses keys in Tap Tap Loot.
## Requirements

- Python 3.x
- Bongo Cat (Steam)

## Installation
Install the required libraries:
```
pip install pydirectinput pywin32 keyboard
```
## Usage

- Open Tap Tap Loot
- Run the script
- Press + to start/stop the bot
- Close the window to exit

## Notes

- Tap Tap Loot must be open before pressing Start
- You can't use your PC normally while the bot is running since Bongo Cat needs to stay in focus
